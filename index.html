🔥 Firebase 연동 설정 가이드

아래 단계를 따라 Firebase를 설정하면 팀원 전체가 같은 URL에서 실시간으로 데이터를 공유할 수 있습니다.

1
Firebase 프로젝트 생성
console.firebase.google.com 접속 → "프로젝트 추가" → 프로젝트명 입력 (예: creative-report)
2
Realtime Database 활성화
좌측 메뉴 → "빌드" → "Realtime Database" → "데이터베이스 만들기" → 테스트 모드로 시작
3
웹 앱 등록 및 설정 복사
프로젝트 설정 → "앱 추가" → 웹(</>) 선택 → 앱 닉네임 입력 → firebaseConfig 값 복사
4
아래 입력창에 붙여넣기 후 저장
복사한 값을 아래 각 항목에 입력하고 "설정 저장" 클릭
